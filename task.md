# 작업: 쿠팡 파트너스 (Coupang Partners) 연동

## 개요
카카오 애드핏에 이어 **쿠팡 파트너스** 광고 배너를 웹사이트에 추가합니다.
쿠팡 파트너스는 사용자가 생성한 **다이나믹 배너 HTML/Script**가 필요합니다.

## 진행 상황
- [x] 카카오 애드핏 연동 완료
- [x] 쿠팡 파트너스 배서 삽입 위치 생성 (`docs/index.html`)
- [x] 사용자 파트너스 코드 확인 및 적용
- [x] GitHub Actions 빌드 오류 수정 (main_site 서브모듈 제거)

## 사용자 할 일
1. **쿠팡 파트너스 접속**: [쿠팡 파트너스 관리자](https://partners.coupang.com/) 로그인.
2. **배너 생성**:
   - 상단 메뉴 [링크 생성] -> [다이나믹 배너] 선택.
   - 원하는 크기 설정 (예: 너비 100%, 높이 자유) 후 `소스 생성` 클릭.
   - **HTML 소스** 복사.
3. **코드 전달**: 복사한 코드를 채팅창에 붙여넣어 주시면 `index.html`에 적용해 드립니다.
